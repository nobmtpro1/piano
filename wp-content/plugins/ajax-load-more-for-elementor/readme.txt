== Description ==
**Ajax Load More for Elementor** improves user experience and gives them a chance to view more of your content with an Ajax powered Load More Button. Instead of loading a whole new page, The Elementor Load More Button works like infinite scroll. It uses Ajax to quickly fetch the next set of content. You can display your posts by clicking on the read more buttton.
